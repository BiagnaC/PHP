Ejercicios Básicos de PHP

Este repositorio contiene una colección de ejercicios básicos desarrollados en PHP, con el objetivo de ayudar a comprender 
la sintaxis, estructura y fundamentos del lenguaje.

Objetivo del Proyecto

El propósito principal es aprender y practicar PHP desde cero, cubriendo conceptos esenciales como:

Variables y tipos de datos
Operadores
Estructuras de control (if, else, switch)
Bucles (for, while, foreach)
Funciones
Manejo de formularios
Arreglos
Conceptos iniciales de programación backend
Estos ejercicios están diseñados tanto para principiantes como para personas que quieren reforzar la lógica en PHP.
